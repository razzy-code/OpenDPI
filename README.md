# OpenDPI X - Özellikler ve Yapılandırma

## 🚀 Genel Bakış

OpenDPI X, gelişmiş DPI atlatma ve bypass teknolojilerini sunan kapsamlı bir masaüstü uygulamasıdır. Bu doküman, uygulamanın tüm özelliklerini ve nasıl yapılandırılacağını açıklamaktadır.

## 📋 İçindekiler

### 🔧 Temel Özellikler
- **Çoklu DPI Atlatma**: -5, -6, -7, -8, -9 metotları ile TTL manipülasyonu
- **DNS Yönlendirme**: Özel DNS sunucularına trafik yönlendirme
- **Otomatik Yeniden Bağlanma**: Kopan bağlantıları otomatik olarak yeniden kurma
- **Discord Rich Presence**: Discord durumunu gösterme
- **Performans İzleme**: Sistem kaynakları ve ağ performansını izleme
- **Türkçe Dil Desteği**: Tam Türkçe arayüzü ve bildirimler
- **Otomatik Güncelleme**: Yeni sürümleri otomatik olarak indirip kurma
- **Hata Raporlama**: Çökmeleri otomatik olarak analiz etme ve raporlama
- **Sistem Temizliği**: Geçici dosyaları ve eski logları otomatik temizleme

### 🚀 Gelişmiş Özellikler (v2.0.0+)

#### 🧠 1. Akıllı Otomatik Metot Seçimi (AI / Adaptive Engine)
- **Gerçek zamanlı analiz**: Ping, packet loss, jitter ölçümleri
- **Otomatik metot değişimi**: En iyi performansı sunan metoda geçiş
- **Öğrenen sistem**: Kullanıcı alışkanlıklarına göre optimizasyon
- **Uygulama algılama**: Discord → Discord profili, Oyun → Gaming profili
- **Akıllı öneriler**: Kullanıcı davranışlarına göre metot önerileri

#### 📊 2. Gerçek Zamanlı Ağ Analiz Paneli (PRO LEVEL)
- **Canlı grafikler**: Anlık ping grafiği
- **Detaylı metrikler**: Packet loss %, jitter, bandwidth (upload/download)
- **Coğrafi analiz**: Ülke bazlı latency haritası 🌍
- **Route optimizasyonu**: "En iyi route" önerileri
- **Wireshark entegrasyonu**: Gelişmiş paket analizi

#### 🌐 3. Multi-Route / Smart Routing Sistemi
- **Trafik bölme**: Discord → Route A, Oyun → Route B, Tarayıcı → Route C
- **Akıllı yönlendirme**: En hızlı yolu otomatik seçme
- **VPN üst seviyesi**: Geleneksel VPN'lerden daha gelişmiş routing
- **Load balancing**: Çoklu bağlantı noktası kullanımı

#### 🔌 4. Plugin Marketplace (EN BÜYÜK OLAY)
- **Açık API**: Kullanıcıların plugin yazabilmesi
- **Marketplace**: Oyun profilleri, özel DNS paketleri, anti-lag modülleri
- **Topluluk desteği**: Kullanıcıların plugin paylaşımı
- **Platform dönüşümü**: OpenDPI X = platform (uygulama değil)

#### 🎮 5. Oyun Algılama Sistemi (Auto Detection)
- **Otomatik tespit**: PC'de çalışan oyunları algılama
- **Profil değişimi**: CS2 → Low latency mod, Roblox → özel profil
- **Oyun optimizasyonu**: Her oyun için özel ayarlar
- **FPS boost**: Oyun performansını artırma

#### ⚡ 6. "One Click Boost" (Bas → Her Şey Optimize)
- **Sistem optimizasyonu**: Gereksiz servisleri kapatma
- **DNS optimizasyonu**: En hızlı DNS sunucularını seçme
- **TCP tweak**: Ağ parametrelerini optimize etme
- **Cache temizleme**: Sistem önbelleğini temizleme
- **Tek tuş = FPS + ping boost**

#### 🛡️ 7. Anti-DPI Shield (Gerçek Zamanlı)
- **DPI tespiti**: Anlık DPI tespiti ve analizi
- **Otomatik karşı metot**: DPI'ya göre otomatik metot seçimi
- **Detaylı loglama**: `[!] DPI detected → switching method (-9)`
- **Proaktif koruma**: DPI engellemelerinden önce önlem alma

#### 📡 8. Built-in Proxy / Tunnel Sistemi
- **SOCKS5/HTTP proxy desteği**: Çoklu protokol desteği
- **Local tunnel**: Yerel tunnel oluşturma
- **Port forwarding**: Gelişmiş port yönlendirme
- **Proxy zinciri**: Çoklu proxy kullanımı

#### 📁 9. Profil Sistemi (Export / Import)
- **Profil kaydetme**: `.opendpi-profile` formatında kayıt
- **Paylaşım**: "Ultra Gaming Profile", "School Bypass Profile"
- **İthalat**: Başka kullanıcılardan profil alma
- **Sürüm kontrolü**: Profil uyumluluğu kontrolü

#### 📊 10. Leaderboard / İstatistik Sistemi
- **Kullanıcı sıralaması**: En düşük ping, en çok uptime
- **Global istatistikler**: Dünya genelinde performans karşılaştırması
- **Başarı rozetleri**: Başarımlara göre rozetler
- **Rekabet hissi**: Hafif rekabet ortamı

#### 🎨 11. Tema Sistemi (NEXT LEVEL UI)
- **Gelişmiş temalar**: Neon, Cyberpunk, Minimal, Liquid Glass
- **Theme marketplace**: Kullanıcıların tema oluşturup paylaşması
- **Canlı animasyonlar**: Dinamik arayüz efektleri
- **Özelleştirme**: Tam arayüz kişiselleştirme

#### 🔐 12. Advanced Security Mode
- **DNS leak protection**: DNS sızıntılarını engelleme
- **IPv6 kontrolü**: IPv6 güvenliği
- **Kill switch**: Bağlantı kesilirse interneti kapatma
- **Şifreli trafik**: End-to-end şifreleme

#### 📢 13. Bildirim Sistemi (Akıllı)
- **Akıllı bildirimler**: "Ping yükseldi", "DPI tespit edildi"
- **Durum güncellemeleri**: "En iyi metot bulundu"
- **Özelleştirilebilir**: Bildirim türlerini ayarlama
- **Sessiz mod**: Gereksiz bildirimleri engelleme

### 🎯 Hedef Kitlemler
- **Oyuncular**: Düşük gecikme ve stabil bağlantı
- **Discord Kullanıcıları**: Sorunsuz ses ve video iletimi
- **Genel Kullanıcılar**: Güvenilir ve hızlı internet erişimi

### 🎮 Metot Profilleri

#### 🎯 Genel Metotlar
- **Standart Mod (Metod -5)**
  - TTL: 5
  - DNS: 77.88.8.8:1253, 2a02:6b8::feed:0ff:1253
  - Açıklama: TTL manipülasyonu + DNS yönlendirme. Stabil ve yaygın.

- **Gelişmiş DPI Atlatma (Metod -1 + Advanced)**
  - TTL: 6
  - DNS: 77.88.8.8:1253, 2a02:6b8::feed:0ff:1253
  - Açıklama: Sert DPI ve engelli siteler için gelişmiş bypass.

- **Ultra Güvenlik Modu (Metod -9 Full)**
  - TTL: 9
  - DNS: 77.88.8.8:1253, 2a02:6b8::feed:0ff:1253
  - Açıklama: Maksimum bypass ve agresif koruma modu.

#### 🎮 Oyun Optimizasyonları
- **Roblox Gaming V1**: -6 + Auto TTL
- **Roblox Low Latency**: -8 (düşük gecikme odaklı)
- **Roblox Ultra Gaming**: -9 Full (maksimum performans)

#### 🎵 Discord Optimizasyonları
- **Discord Optimize V1**: -9 + TTL 3
- **Discord Streaming Boost**: -9 (stream ve ekran paylaşımı odaklı)
- **Discord Ultra Bypass**: -1 Advanced (yüksek uyumluluk)

#### 🌐 Tarayıcı Optimizasyonları
- **Tarayıcı Turbo V1**: -5 (genel web erişimi)
- **Engelli Site Bypass**: -7 (engelli siteler için)
- **Ultra Site Geçiş**: -9 Advanced (maksimum site erişimi)

#### 🔧 Servis Metotları

## 🛠️ Geliştirme Araçları

### 🔧 Komut Satırı Arayüzü
Geliştiriciler için komut satırı arayüzü:
```bash
# Sistem temizliği
opendpi-x --cleanup

# Performans izleme
opendpi-x --monitor

# Ağ teşhis
opendpi-x --diagnose

# Yapılandırma yönetimi
opendpi-x --config --list
opendpi-x --config --set <parametre> <değer>

# Servis yönetimi
opendpi-x --service --start <metot>
opendpi-x --service --stop
opendpi-x --service --restart
opendpi-x --service --status

# AI destekli özellikler (v2.0.0+)
opendpi-x --ai --adaptive-method
opendpi-x --ai --optimize-route
opendpi-x --ai --detect-games

# Gelişmiş analiz
opendpi-x --analyze --real-time
opendpi-x --analyze --packet-loss
opendpi-x --analyze --bandwidth

# One Click Boost
opendpi-x --boost --all
opendpi-x --boost --gaming
opendpi-x --boost --streaming

# Anti-DPI Shield
opendpi-x --shield --enable
opendpi-x --shield --detect

# Proxy/Tunnel sistemi
opendpi-x --proxy --start <type> <port>
opendpi-x --tunnel --create <port>

# Profil yönetimi
opendpi-x --profile --export <dosya>
opendpi-x --profile --import <dosya>

# Plugin sistemi
opendpi-x --plugin --list
opendpi-x --plugin --install <dosya>
opendpi-x --plugin --remove <isim>

# Tema sistemi
opendpi-x --theme --list
opendpi-x --theme --set <isim>
opendpi-x --theme --install <dosya>
```

### ❓ Sıkça Sorulan Sorular
**S: OpenDPI X çalışmıyor?**
C: Çalıştırma komutunu yönetici olarak çalıştırın: `opendpi-x --service --start "Standart Mod (Metod -5)"`

**S: Discord RPC çalışmıyor?**
C: Discord uygulamasının açık olduğundan emin olun. Ayarlardan Discord entegrasyonunu devre dışı bırakıp yeniden etkinleştirin.

**S: Bağlantı çok yavaş?**
C: Farklı bir metot deneyin veya DNS sunucularını değiştirin. Ağ teşhis aracını kullanın: `opendpi-x --diagnose`

**S: Güncelleme indirilemiyor?**
C: İnternet bağlantınızı kontrol edin. Güvenlik duvarını devre dışı bırakıp tekrar deneyin.

**S: Servis çöküyor?**
C: Sistem kaynaklarınızı kontrol edin. Yüksek CPU kullanımı varsa daha az kaynaklı bir metot seçin.
